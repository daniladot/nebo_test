1) Для построения маршрута до МКАДа щелкните по любой точке на карте
2) Для перестроения маршрута снова щелкните по любой точке на карте
3) Для отображения адреса, расстояния по воздуху и дороге повторно нажмите на заданную точку

Данная программа работает на основе построения ближайших точек на МКАДе по прямой и строит к ним 
кратчайший маршрут от заданной точки.
